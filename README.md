# Hey there!

#### Thank you for visiting my github page for my [Frontend Mentor](https://www.frontendmentor.io) solutions.

#### Frontend Mentor is a site that has numerous challenges ranging from newbie to expert level.  These give developers an opportunity to practice and hone their skills with HTML, CSS and JS.


## What are the challenges?

#### Each challenge gives you a set of images that shows what each challenge looks like when completed.  The goal is to recreate each image and add any features mentioned and make it look as close as possible to the images.  No building blocks or templates are given, just the images needed for the challenge.

Each of the folders will have a README file that goes over each challenge and details my journey during each one.  Thanks for joining me on these challenges and enjoy the ride.
